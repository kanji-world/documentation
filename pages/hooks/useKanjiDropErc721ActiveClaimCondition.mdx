# useKanjiDropErc721ActiveClaimCondition

This hook retrieves the active ERC721 claim condition, based on the provided address, and returns an object containing its loading and error states, whether or not it has been successfully retrieved, and its error object, if any.

## Usage

```tsx
import { useKanjiDropErc721ActiveClaimCondition } from '@kanji-world/contracts-react'

const { claimCondition, isLoading, isSuccess, isError, error } = useKanjiDropErc721ActiveClaimCondition({
  address,
})
```


## Return Value

| Property        | Type      | Description                                       |
| --------------- | --------- | ------------------------------------------------- |
| `claimCondition` | `object`  | An object containing the active ERC721 claim condition. |
| `isLoading`     | `boolean` | A boolean indicating whether the claim condition is currently loading|
| `isSuccess`     | `boolean` | A boolean indicating whether the claim condition has been successfully retrieved|
| `isError`       | `boolean` | A boolean indicating whether an error has occurred while retrieving the claim condition |
| `error`         | `Error`   | An Error object containing errors from the claim condition retrieval process. |


## Configuration

| Property       | Type                                                      | Description                                                |
| ---------------| --------------------------------------------------------- | -------------------------------------------------- |
| `address`      | `string`                                                  | Required. The address of the ERC721 contract |
| `editionId`    | `BigNumberish`                                            | Optional. The edition ID of the ERC721 contract. |
