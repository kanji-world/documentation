
# useKanjiDropErc1155IsWhitelisted

React hook that can be used to retrieve the whitelisting status of an account for a specific ERC1155 claim condition.
## Usage
```jsx
import { useKanjiDropErc1155IsWhitelisted } from '@kanji-world/react-sdk'

const MyComponent = () => {
  const { isWhitelisted, isLoading, isError, error } = useKanjiDropErc1155IsWhitelisted({
    address: '0x123...',
    account: '0xabc...',
    claimConditionId: 1
  })

  // render component using the whitelisting status...

  return (
    ...
  )
}
```

## Return Value
`useKanjiDropErc1155IsWhitelisted` returns an object with the following properties:
- `isWhitelisted`: a boolean indicating whether the specified account is whitelisted for the specified ERC1155 claim condition.
- `isLoading`: a boolean indicating whether the whitelisting status is currently loading.
- `isSuccess`: a boolean indicating whether the whitelisting status has been successfully retrieved.
- `isError`: a boolean indicating whether an error has occurred while retrieving the whitelisting status.
- `error`: an Error object containing errors from the whitelisting status retrieval process.

## Configuration
`useKanjiDropErc1155IsWhitelisted` accepts an object with the following properties:
- `address`: The address of the ERC1155 contract.
- `account`: The account being checked for whitelisting.
- `claimConditionId`: The ID of the claim condition to be checked.
