# useKanjiDropErc721TokenMetadata

React Hook that returns the metadata associated with a specified ERC721 token.

## Usage

```tsx
import { useKanjiDropErc721TokenMetadata } from '@kanji-world/react-sdk'
```

```tsx
const { metadata, isLoading, isSuccess, isError, error } = useKanjiDropErc721TokenMetadata({
  address: '0x...',
  tokenId: 1
})
```

## Return Value

| Property  | Type                | Description                                                       |
| --------- | ------------------- | ----------------------------------------------------------------- |
| `metadata`  | `{ name: string, description: string, image: string, attributes: { trait_type: string; value: string }[] }` | The metadata associated with the ERC721 token.             |
| `isLoading` | `boolean` | `true` when data is being fetched, `false` otherwise.                  |
| `isSuccess` | `boolean` | `true` when data fetching is successful, `false` otherwise. |
| `isError`   | `boolean` | `true` when an error occurred during data fetching, `false` otherwise.         |
| `error`     | `Error`             | The error object with the relevant error message.          |

## Configuration

| Property | Type     |  Description                                                                   |
| -------- | -------- |  ----------------------------------------------------------------------------- |
| `address`   | `Address`  | The address of the ERC721 contract being queried.                   |
| `tokenId`  | `BigNumberish` |The ID of the ERC721 token being queried.|
